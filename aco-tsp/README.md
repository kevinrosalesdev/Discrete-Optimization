# ACO - TSP

Problema de TSP solucionado con distintos algoritmos:

Implementaciones y ejecución de éstas:
```
Greedy =>
python -m greedy -p [rutaDelArchivo]
2-Approx =>
python -m 2approx -p [rutaDelArchivo]
Christofides =>
python -m christo -p [rutaDelArchivo]
2Opt con Christofides =>
python -m 2opt_christo -p [rutaDelArchivo]
2Opt con 2-Approx =>
python -m 2opt_2approx -p [rutaDelArchivo]
3Opt con Christofides =>
python -m 3opt_christo -p [rutaDelArchivo]
3Opt con 2-Approx =>
python -m 3opt_2approx -p [rutaDelArchivo]
Simulated Annealing =>
python -m simua -p [rutaDelArchivo]
Simulated Annealing + Tabu Search =>
python -m simutabu -p [rutaDelArchivo]
Genetic Algorithm =>
python -m genetic -p [rutaDelArchivo]
Ant System =>
python -m ant -p [rutaDelArchivo]
```
