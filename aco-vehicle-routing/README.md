**[Algoritmo de Vehicle Routing en Python]**

Este algoritmo incluye:

- Algoritmo Greedy.
- Algoritmo usando MIP.

Para ejecutar el algoritmo:
```
Greedy => 
python solver.py -m greedy -p [rutaDeArchivo]
MIP => 
python solver.py -m mip -p [rutaDeArchivo]
```
