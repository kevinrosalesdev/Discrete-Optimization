# Discrete-Optimization

Some Problems solved in order to learn Discrete Optimization Techniques.

## List of problems
- 8-Queens
- Facility Location
- Graph Coloring
- Knapsack
- Magic Square
- Set Covering
- TSP
- Vehicle Routing

All problems are solved using Python and some libraries such as Gurobi.
